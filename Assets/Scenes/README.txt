# Scene Setup Instructions
# BoxReha Game

This folder contains placeholder instructions for creating Unity scenes.

## Required Scenes

Create these 4 scenes in Unity:

1. **Boot.unity** - Bootstrap scene (loads first)
2. **MainMenu.unity** - Main menu with Start/Quit buttons
3. **Game.unity** - Main gameplay scene
4. **Results.unity** - Results/stats screen

## Quick Setup

See SETUP_GUIDE.md in the project root for complete step-by-step instructions.

## Scene Build Order

In File → Build Settings, add scenes in this order:
- 0: Boot
- 1: MainMenu
- 2: Game
- 3: Results

## Note

These are placeholder files. You must create the actual .unity scene files
in the Unity Editor following the SETUP_GUIDE.md instructions.
